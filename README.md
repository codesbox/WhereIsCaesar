# WhereIsCaesar - Поиск ресторанов по рейтингам блюд

### [---Видео---](https://www.youtube.com/watch?v=jCkAoZnNlAQ)

### Описание
Я думаю, у каждого бывала ситуация, что хочется пойти в ресторан, чтобы попробовать свое любимое блюдо. Или же вы уже пришли в ресторан и у вас разбегаются глаза от обширности меню. Но что выбрать? Какое блюдо действительно будет самым вкусным из этого меню? На помощь приходит WhereIsCaesar. Я предлагаю кардинально новую систему оценки ресторанов - по блюдам. У каждого ресторана есть список оцененных блюд. Соответственно, при клике на ресторан можно посмотреть список оцененных блюд, их рейтинг, отзывы на каждое блюдо по отдельности. Также вы можете искать рестораны по составному запросу из блюд, которые вы хотите поесть. Вам будут показаны только те рестораны, в которых есть все ваши выбранные блюда.
